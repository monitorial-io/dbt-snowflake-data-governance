## Context

This is a **macro-only dbt package** (`dbt_monitorial_datagovernance`) providing post-hook macros for applying Snowflake data governance features (tags, aggregation policies, projection policies, row access policies). It has 10 commits, 4 version tags (0.1.0-0.1.3), but the `dbt_project.yml` version is stale at `0.1.0`.

- **No tests** of any kind exist
- **No CI/CD pipelines** exist
- **Several bugs** in macros (undefined variables, wrong `selectattr` index, dead code)
- **Version mismatch**: `dbt_project.yml` says `0.1.0`, CHANGELOG is at `0.1.3`
- **Deprecated configs**: `target-path` and `clean-targets` are deprecated in dbt-core 1.5+
- **README errors**: wrong macro name in Projection Policies header, stale install revision
